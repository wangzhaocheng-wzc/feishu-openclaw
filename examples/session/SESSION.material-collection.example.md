# SESSION.md (示例) - 收集材料

## 当前目标

- 为某个主题收集高质量材料（覆盖面、可信度、可引用性），并整理成可用的摘要/要点/链接库。

## 关键决策（含理由）

- 先定义“收集范围”和“排除范围”。
  - 理由：防止无限扩散。
- 每条材料必须记录：来源链接 + 可信度判断 + 可引用片段。
  - 理由：后续写作/汇报可追溯。

## 明确否决的方案（含理由）

- 不把未经核实的二手总结当作结论。
  - 理由：容易传播错误信息。
