# 交付模板（可复用）

这组模板用于把三类常见任务的最终交付物标准化：写文章、收集材料、接入新技能。

使用方式：

- 写作任务：复制 `templates/DELIVERABLE.article.md`
- 材料收集：复制 `templates/DELIVERABLE.material-pack.md`
- 技能接入：复制 `templates/DELIVERABLE.skill-integration.md`

建议流程：

1) 用 `SESSION.md` 管过程
2) 交付时用对应 `DELIVERABLE.*` 模板输出最终结果
3) 有可复用资产（docs/templates/scripts/config）就一起落库并 commit
